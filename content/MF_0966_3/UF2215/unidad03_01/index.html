    Unidad 3: Arrays, Objetos y Clases

    3.1 Arrays y su Manipulación

    ¿Qué es un Array?

    Un array es una colección de elementos que se almacenan en un orden determinado. Podemos pensar en un array como una lista de elementos, similar a una colección de frutas, una lista de ingredientes de una receta o una lista de tareas pendientes.

    Ejemplo de lista de frutas:

    Manzana, Banana, Cereza

    Si añadimos una fruta:

    Manzana, Banana, Cereza, Naranja

    Si eliminamos un elemento:

    Manzana, Cereza, Naranja

    Si modificamos un elemento:

    Manzana, Uva, Cereza, Naranja

    Los arrays permiten agregar, eliminar y modificar elementos de manera sencilla. Dependiendo de cómo realicemos estas operaciones, la estructura del array cambiará. Por ejemplo:

    • Añadir elementos al final mantiene el orden y simplemente extiende la lista.
    • Añadir elementos en el medio desplaza los elementos existentes.
    • Eliminar elementos puede dejar huecos si no se maneja correctamente.

    function buscarIndice(array, valor) {
    for (let i = 0; i < array.length; i++) {
        if (array[i] === valor) {
            return i;
        }
    }
    return -1;
}

let frutas = ["Manzana", "Banana", "Cereza", "Naranja"];
console.log(buscarIndice(frutas, "Cereza")); // 2
console.log(buscarIndice(frutas, "Pera")); // -1

    3.1.7 Métodos Básicos de Arrays

    JavaScript proporciona varios métodos para manipular arrays de manera eficiente. A continuación, se presentan algunos de los más utilizados:

    Métodos de Verificación

    • Array.isArray(): Verifica si una variable es un array.
      Ejemplo de Array.isArray()
      1
2

console.log(Array.isArray("Hola")); // false

    Métodos de Ordenación y Conversión

    • sort(): Ordena los elementos del array.

      Ejemplo de sort()
      1
2
3

    • reverse(): Invierte el orden de los elementos.

      Ejemplo de reverse()

2

console.log(frutas); // ["Manzana", "Cereza", "Banana"]

    • join(): Convierte el array en una cadena con un separador.

      Ejemplo de join()
      1
      console.log(frutas.join(", ")); // "Manzana, Cereza, Banana"

    Pilas y Colas en Arrays

    Los arrays pueden comportarse como pilas (stack) o colas (queue) dependiendo de cómo se manejen los elementos.

    • Pilas (Stack): Siguen el principio LIFO (Last In, First Out). Se manejan con push() (añadir al final) y pop() (eliminar del final).
    • Colas (Queue): Siguen el principio FIFO (First In, First Out). Se manejan con push() (añadir al final) y shift() (eliminar del inicio).
